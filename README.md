# 🛍️ Customer Segmentation using K-Means Clustering

An end-to-end machine learning project that segments customers into distinct groups based on demographics, spending behavior, and purchase patterns.

## 🚀 Tech Stack
- Python 3.10+
- Pandas & NumPy
- Matplotlib & Seaborn
- Scikit-learn

## 📦 Features
- Full EDA with visualizations
- Correlation heatmap
- Elbow Method to find optimal K
- KMeans clustering into 4 segments
- Outputs saved as CSV

## 🎯 Cluster Insights
- Cluster 0 — Low income, low spend (budget shoppers)
- Cluster 1 — High income, high spend (premium customers)
- Cluster 2 — Mid income, frequent web buyers (digital natives)
- Cluster 3 — Older, high recency (at-risk customers)
